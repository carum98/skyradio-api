# Update Radio Model

## Role allowed
`admin`

## URL
`/radios-model/:code`

## Method
`PUT`

## URL Params
| Name | Type | Description |
| --- | --- | --- |
| code | string | Radio model code |

## Data Params
| Name | Type |
| --- | --- |
| name | string |
| color | string |

```json
{
    "name": "Radio model name",
    "color": "#000000"
}
```
